# 0017 灰度版产品边界

日期：2026-05-05

## 决策

小票夹灰度版必须按普通生活记账 App 设计，不能把服务器、接口和运维诊断作为普通用户主体验。

普通用户默认看：

- 账本名。
- 连接状态。
- 最近上传。
- 最近同步。
- 上传截图。
- 待确认。
- 账本。
- 统计。
- 外观。
- 数据与安全。

普通用户默认不看：

- 服务器域名。
- token。
- 接口名。
- Cloudflare。
- 端口。
- 日志。
- Windows 计划任务。
- 后端进程。
- 诊断脚本。

## 原因

灰度用户的服务器不在他那里。给普通用户看服务器诊断会制造困惑，也会暴露不必要的技术信息。

## 影响

- Android 设置页要去开发味。
- 运维诊断留在 Windows 脚本和 docs。
- 后续 Web 管理页可以承接服务拥有者功能。
- Release 版不显示开发诊断入口。
