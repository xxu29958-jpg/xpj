## 设计红线

- 不设计银行聚合、信用卡同步、投资净资产或真实资产负债表。
- 不设计自动入账；截图和 OCR 草稿仍必须人工确认。
- 不把 `/owner` 设计成公网普通用户入口。
- 图表设计不能替代分页、聚合、账本隔离和权限边界。
- 普通用户 UI 不展示 token、UploadLink、Cloudflare、端口、接口名、本机路径或日志。
